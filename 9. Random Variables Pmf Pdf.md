# Random Variables, Discrete & Continuous Variables, PMF and PDF

---

## 1. Random Variable

### Simple Meaning

A **Random Variable** is a variable that represents the **numerical outcome of a random experiment**.

Instead of writing outcomes as words, we assign **numbers** to them.

So a random variable **maps outcomes → numbers**.

---

### Example 1: Tossing a Coin

Experiment: Toss a coin.

Possible outcomes:

| Outcome | Random Variable (X) |
| ------- | ------------------- |
| Head    | 1                   |
| Tail    | 0                   |

Here **X** is the **random variable**.

Possible values of **X = {0,1}**

---

### Example 2: Rolling a Dice

Experiment: Roll a dice.

Possible outcomes:

1,2,3,4,5,6

Define random variable **X = number on the dice**

Possible values:

**X = {1,2,3,4,5,6}**

---

### Key Idea

A **Random Variable converts experiment outcomes into numbers** so that we can analyze them mathematically.

---

# 2. Types of Random Variables

There are **two types**:

1. **Discrete Random Variable**
2. **Continuous Random Variable**

---

# 3. Discrete Random Variable

### Meaning

A **Discrete Random Variable** can take **countable values**.

These values are usually **integers**.

You can **list all possible values**.

---

### Examples

| Example                     | Possible Values |
| --------------------------- | --------------- |
| Number of students in class | 0,1,2,3…        |
| Number of cars passing      | 0,1,2,3…        |
| Number on dice              | 1,2,3,4,5,6     |

---

### Example: Tossing Two Coins

Possible outcomes:

HH, HT, TH, TT

Let random variable **X = number of heads**

| Outcome | X |
| ------- | - |
| HH      | 2 |
| HT      | 1 |
| TH      | 1 |
| TT      | 0 |

Possible values:

**X = {0,1,2}**

These are **countable**, so it is **discrete**.

---

### Key Idea

Discrete variables represent **counts**.

Examples:

* Number of emails
* Number of customers
* Number of defects

---

# 4. Continuous Random Variable

### Meaning

A **Continuous Random Variable** can take **any value within a range**.

There are **infinitely many possible values**.

---

### Examples

| Variable           | Possible Values         |
| ------------------ | ----------------------- |
| Height of a person | 150.1, 150.12, 150.123… |
| Weight             | 60.1, 60.12…            |
| Temperature        | 25.1, 25.12…            |
| Time               | 1.2 sec, 1.23 sec       |

These values are **not countable**.

---

### Example

Suppose

**X = height of students**

Possible values:

150.1
150.25
150.256
150.2568

There are **infinite possible values between 150 and 151**.

So it is **continuous**.

---

### Key Idea

Continuous variables represent **measurements**.

Examples:

* Height
* Weight
* Time
* Temperature
* Speed

---

# 5. Probability Mass Function (PMF)

Used for **Discrete Random Variables**.

---

### Meaning

A **Probability Mass Function (PMF)** gives the **probability of each possible value** of a discrete random variable.

It tells us:

```
P(X = x)
```

Probability that random variable **X takes value x**.

---

### Example: Rolling a Dice

Let **X = number on the dice**

Possible values:

1,2,3,4,5,6

Probability of each value:

| X | P(X) |
| - | ---- |
| 1 | 1/6  |
| 2 | 1/6  |
| 3 | 1/6  |
| 4 | 1/6  |
| 5 | 1/6  |
| 6 | 1/6  |

This table is the **PMF**.

---

### Properties of PMF

1. Probabilities are **between 0 and 1**

```
0 ≤ P(X = x) ≤ 1
```

2. Total probability must equal **1**

```
Σ P(X = x) = 1
```

---

### Example

Suppose

| X | P(X) |
| - | ---- |
| 0 | 0.2  |
| 1 | 0.5  |
| 2 | 0.3  |

Check:

```
0.2 + 0.5 + 0.3 = 1
```

So it is a **valid PMF**.

---

# 6. Probability Density Function (PDF)

Used for **Continuous Random Variables**.

---

### Meaning

For continuous variables, we cannot calculate probability at **exact points**.

Example:

Probability that height = **170.000000 cm** exactly is **0**.

Instead we calculate probability over **ranges**.

So we use a **Probability Density Function (PDF)**.

It describes **how probability is distributed over a range of values**.

---

### Example

Suppose

**X = height of students**.

We calculate probability like:

```
P(160 < X < 170)
```

Meaning:

Probability that height is **between 160 and 170 cm**.

---

### Key Difference

Discrete:

```
P(X = 3)
```

Continuous:

```
P(3 < X < 5)
```

---

### Properties of PDF

1. PDF values are **non‑negative**

```
f(x) ≥ 0
```

2. Total area under curve = **1**

```
∫ f(x) dx = 1
```

---

# 7. PMF vs PDF (Very Important)

| Feature     | PMF                | PDF                  |
| ----------- | ------------------ | -------------------- |
| Used for    | Discrete variables | Continuous variables |
| Values      | Countable          | Infinite             |
| Probability | Exact values       | Range of values      |
| Example     | Dice               | Height               |

---

### Example

Discrete:

Number of customers arriving

```
P(X = 3)
```

Continuous:

Temperature

```
P(20 < X < 25)
```

Examples:

Think like this:

Random Variable
→ converts outcomes into numbers

Discrete Variable
→ countable values (0,1,2,3)

Continuous Variable

PMF

PDF
