# Probability Mass Function (PMF) and Probability Density Function (PDF)

---

# 1. Introduction

In probability and statistics, we often want to describe **how probability is distributed across possible outcomes of a random variable**.

Two fundamental functions are used for this purpose:

| Function                               | Used For                    |
| -------------------------------------- | --------------------------- |
| **PMF (Probability Mass Function)**    | Discrete Random Variables   |
| **PDF (Probability Density Function)** | Continuous Random Variables |

Understanding these concepts is extremely important in:

* Statistics
* Machine Learning
* Data Science
* Risk Modelling
* Financial Modelling

---

# 2. Random Variables

A **random variable** is a variable whose value depends on the outcome of a random experiment.

### Examples

| Experiment       | Random Variable |
| ---------------- | --------------- |
| Tossing a coin   | Number of heads |
| Rolling a dice   | Dice value      |
| Measuring height | Person height   |
| Measuring time   | Waiting time    |

Random variables are generally classified into **two types**.

---

# 3. Discrete vs Continuous Variables

## Discrete Random Variable

A **discrete random variable** can take **countable values**.

Examples:

* Number of students in a class
* Dice outcomes (1,2,3,4,5,6)
* Number of emails received today

Example values:

```
0,1,2,3,4,5...
```

## Continuous Random Variable

A **continuous random variable** can take **any value within a range**.

Examples:

* Height
* Weight
* Temperature
* Time

Example values:

```
170.1
170.11
170.111
170.1115
```

There are **infinitely many values**.

---

# 4. Probability Mass Function (PMF)

## Definition

A **Probability Mass Function (PMF)** gives the probability that a **discrete random variable** takes a specific value.

Mathematically:

```
P(X = x)
```

Meaning:

Probability that random variable **X equals a specific value x**.

---

# 5. PMF Examples

## Example 1: Dice Roll

Let

```
X = outcome of a fair dice
```

Possible values:

| X | Probability |
| - | ----------- |
| 1 | 1/6         |
| 2 | 1/6         |
| 3 | 1/6         |
| 4 | 1/6         |
| 5 | 1/6         |
| 6 | 1/6         |

PMF:

```
P(X = x) = 1/6
```

for

```
x = 1,2,3,4,5,6
```

---

## Example 2: Two Coin Tosses

Let

```
X = number of heads
```

Possible outcomes:

```
HH
HT
TH
TT
```

Probability table:

| Heads (X) | Probability |
| --------- | ----------- |
| 0         | 1/4         |
| 1         | 2/4         |
| 2         | 1/4         |

PMF:

```
P(X=0)=1/4
P(X=1)=1/2
P(X=2)=1/4
```

---

# 6. Properties of PMF

A PMF must satisfy the following conditions.

## 1. Non-negative

```
P(X = x) ≥ 0
```

## 2. Total probability equals 1

```
Σ P(X = x) = 1
```

Example:

```
1/4 + 1/2 + 1/4 = 1
```

## 3. Defined only for discrete values

PMF works only when outcomes are **countable**.

---

# 7. Probability Density Function (PDF)

## Definition

A **Probability Density Function (PDF)** describes the probability distribution of a **continuous random variable**.

Instead of probability at a single point, we compute probability over an **interval**.

Mathematically:

```
P(a ≤ X ≤ b) = ∫ f(x) dx
```

Where:

| Symbol | Meaning                      |
| ------ | ---------------------------- |
| f(x)   | probability density function |
| a,b    | interval limits              |

---

# 8. Important Concept

For continuous variables:

```
P(X = x) = 0
```

Why?

Because there are **infinite possible values**.

Example:

Probability that a person's height is **exactly 170.000000 cm** is essentially **zero**.

Instead we measure probability across **ranges**.

Example:

```
P(169 ≤ height ≤ 171)
```

---

# 9. PDF Example

## Uniform Distribution Example

Suppose waiting time for a bus is between

```
0 to 10 minutes
```

Then PDF is

```
f(x) = 1/10
```

for

```
0 ≤ x ≤ 10
```

### Find Probability

```
P(2 ≤ X ≤ 5)
```

Using integration:

```
P = ∫(2 to 5) 1/10 dx
```

```
= (1/10)(5-2)
```

```
= 3/10
```

```
= 0.3
```

So probability is **0.3**.

---

# 10. Properties of PDF

## 1. Density is non-negative

```
f(x) ≥ 0
```

## 2. Total area under curve equals 1

```
∫(-∞ to ∞) f(x) dx = 1
```

## 3. Probability is area under curve

```
P(a ≤ X ≤ b) = area under curve between a and b
```

---

# 11. PMF vs PDF (Comparison)

| Feature                 | PMF                | PDF                 |
| ----------------------- | ------------------ | ------------------- |
| Variable Type           | Discrete           | Continuous          |
| Probability at value    | Yes                | No                  |
| Formula                 | P(X=x)             | f(x)                |
| Probability calculation | Direct probability | Area under curve    |
| Graph                   | Bars               | Smooth curve        |
| Examples                | Dice, email count  | Height, temperature |

---

# 15. Python Implementation

## PMF Example (Dice)

```python
import numpy as np

values = [1,2,3,4,5,6]
pmf = [1/6]*6

for v,p in zip(values,pmf):
    print(f"P(X={v}) = {p}")
```

Output

```
P(X=1) = 0.166
P(X=2) = 0.166
P(X=3) = 0.166
...
```

---

## PDF Example (Normal Distribution)

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

x = np.linspace(-3,3,100)
pdf = norm.pdf(x,0,1)

plt.plot(x,pdf)
plt.title("Normal Distribution PDF")
plt.show()
```

This produces a **continuous probability density curve**.

---

# 16. Key Takeaways

* **PMF** is used for **discrete random variables**.
* **PDF** is used for **continuous random variables**.
* PMF gives **probability at specific values**.
* PDF gives **probability density across intervals**.
* For continuous variables **P(X=x)=0**.
* Probability in continuous distributions comes from **area under the curve**.
