# 1. Z-Test

## What it is

A **Z-test** checks whether the **mean of a sample is significantly different from the population mean**.

In simple words:

> It tells us whether the difference between a sample and the population happened **by chance or not**.

---

## When to use a Z-Test

Use a Z-test when:

* Sample size is **large (n ≥ 30)**
* Population **standard deviation is known**
* Data is approximately **normally distributed**

---

## Formula

```
Z = (x̄ − μ) / (σ / √n)
```

Where:

| Symbol | Meaning                       |
| ------ | ----------------------------- |
| x̄     | Sample mean                   |
| μ      | Population mean               |
| σ      | Population standard deviation |
| n      | Sample size                   |

---

## Real Example

A company claims that the **average salary of employees is ₹50,000**.

A researcher collects a **sample of 100 employees**.

Sample results:

| Parameter                     | Value   |
| ----------------------------- | ------- |
| Sample mean salary            | ₹53,000 |
| Population standard deviation | ₹10,000 |
| Sample size                   | 100     |

### Question

Is the company's claim correct?

### Hypothesis

**Null Hypothesis (H0)**
Average salary = ₹50,000

**Alternative Hypothesis (H1)**
Average salary ≠ ₹50,000

We perform a **Z-test**.

If the calculated Z value is large enough, the difference is considered **statistically significant**.

Conclusion example:

The sample suggests the **average salary is significantly different from ₹50,000**.

---

# 2. T-Test

## What it is

A **T-test** checks whether **means are significantly different when the sample size is small**.

It is very similar to the Z-test but used when **population standard deviation is unknown**.

---

## When to use a T-Test

Use a T-test when:

* Sample size **less than 30**
* Population standard deviation **unknown**
* Data is approximately normal

---

## Formula

```
t = (x̄ − μ) / (s / √n)
```

Where:

| Symbol | Meaning                   |
| ------ | ------------------------- |
| x̄     | Sample mean               |
| μ      | Population mean           |
| s      | Sample standard deviation |
| n      | Sample size               |

---

## Types of T-Tests

1. **One-Sample T-Test**
2. **Independent T-Test**
3. **Paired T-Test**

---

## Real Example (One-Sample T-Test)

A university claims:

> The **average exam score is 70**.

A teacher randomly selects **15 students**.

Sample results:

| Parameter                 | Value |
| ------------------------- | ----- |
| Sample mean score         | 75    |
| Sample standard deviation | 8     |
| Sample size               | 15    |

### Hypothesis

**H0:** Average score = 70
**H1:** Average score ≠ 70

Since the **sample size is small (n = 15)**, we use a **T-test**.

Conclusion example:

If the T value is large enough, we conclude that **students performed significantly better than the national average**.

# 3. Chi-Square Test

## What it is

The **Chi-square test** checks whether **two categorical variables are related**.

In simple terms:

> It tests whether **observed results are different from expected results**.

It is used with **categorical data**, not numerical averages.

---

## Examples of Categorical Variables

* Gender
* Product choice
* Pass / Fail
* Yes / No

---

## Formula

```
χ² = Σ (O − E)² / E
```

Where:

| Symbol | Meaning        |
| ------ | -------------- |
| O      | Observed value |
| E      | Expected value |

---

## Real Example

A company wants to understand:

> Does **gender influence product choice**?

Survey results:

| Gender | Product A | Product B |
| ------ | --------- | --------- |
| Male   | 40        | 20        |
| Female | 30        | 50        |

Observed values represent the **actual survey data**.

Expected values represent what we would expect **if gender had no influence on product choice**.

Using a **Chi-square test**, we evaluate whether there is a **relationship between gender and product preference**.

### Possible Conclusions

If the Chi-square statistic is large:

* Product preference **depends on gender**

If the Chi-square statistic is small:

* There is **no relationship** between gender and product choice

# 4. ANOVA (Analysis of Variance)

## What it is

**ANOVA** is used when we want to compare **means of three or more groups**.

Instead of running multiple T-tests, ANOVA compares all groups **in a single statistical test**.

---

## Real Example

A company evaluates **three employee training programs**.

Goal:

> Determine which training program improves employee performance the most.

Sample results:

| Training Program | Average Score |
| ---------------- | ------------- |
| Program A        | 75            |
| Program B        | 82            |
| Program C        | 78            |

---

## Hypothesis

**Null Hypothesis (H0)**

All group means are equal

```
μA = μB = μC
```

**Alternative Hypothesis (H1)**

At least **one group mean is different**.

---

## Concept Behind ANOVA

ANOVA compares two types of variation:

| Type of Variation      | Meaning                         |
| ---------------------- | ------------------------------- |
| Between-group variance | Differences between group means |
| Within-group variance  | Variation inside each group     |

If **between-group variance is much larger**, the group means are considered **significantly different**.

---

## Conclusion

If ANOVA produces a significant result:

* At least **one training program performs differently** from the others

To identify which one, we perform **post-hoc tests** such as:

* Tukey Test
* Bonferroni Test

# Quick Comparison Table

| Test       | Purpose                                    | Data Type   | Example                            |
| ---------- | ------------------------------------------ | ----------- | ---------------------------------- |
| Z-Test     | Compare sample mean with population mean   | Numerical   | Sample salary vs population salary |
| T-Test     | Compare means with small sample            | Numerical   | Student score vs national average  |
| Chi-Square | Relationship between categorical variables | Categorical | Gender vs product choice           |
| ANOVA      | Compare means of 3 or more groups          | Numerical   | Training program performance       |

---

# Simple Intuition to Remember

A quick way to remember when to use each test:

* **Z-Test → 1 sample, large dataset**
* **T-Test → small sample size**
* **Chi-Square → categorical data**
* **ANOVA → compare 3 or more group means**
