# Statistics Visualizations: Histogram, Boxplot, and Scatter Plot

---

## 1. Histogram

A **Histogram** is a graph that shows **how data is distributed**.

Instead of showing individual values, it groups values into **ranges (bins)**.

### Example Dataset (Marks of Students)

```
10, 20, 25, 30, 35, 40, 45, 50, 55, 60
```

### Histogram Grouping Example

| Marks Range | Number of Students |
| ----------- | ------------------ |
| 0–20        | 2                  |
| 21–40       | 4                  |
| 41–60       | 4                  |

So a histogram tells us **where most data lies**.

---

# Types of Histogram Shapes

## 1. Symmetric Distribution

A **symmetric histogram** looks like a **bell shape**.

Left side and right side look almost the same.

### Example

Heights of students in a class:

```
150, 155, 158, 160, 162, 165, 168, 170
```

Most students are near the **middle value**.

### Relation between Mean and Median

In symmetric distribution:

```
Mean = Median
```

### Example

Data:

```
2, 4, 6, 8, 10
```

Mean = 6
Median = 6

---

## 2. Right Skewed Distribution (Positive Skew)

In **right skewed data**, the **tail goes to the right**.

Most values are **small**, but a few **very large values** pull the tail right.

### Example: Income of people

Most people earn moderate income, but a few earn extremely high salaries.

### Example Data

```
20, 22, 24, 25, 26, 28, 30, 80
```

### Relation between Mean and Median

Large values pull the mean upward.

```
Mean > Median
```

### Example

Median = 25
Mean = 31 (influenced by 80)

---

## 3. Left Skewed Distribution (Negative Skew)

In **left skewed data**, the **tail goes to the left**.

Most values are **large**, but a few **very small values** pull the tail left.

### Example: Exam scores in an easy exam

Most students score high, but a few fail.

### Example Data

```
30, 70, 75, 80, 85, 88, 90
```

### Relation between Mean and Median

Small values pull the mean downward.

```
Mean < Median
```

---

## Quick Summary

| Distribution | Shape         | Mean vs Median |
| ------------ | ------------- | -------------- |
| Symmetric    | Balanced      | Mean = Median  |
| Right Skewed | Tail on right | Mean > Median  |
| Left Skewed  | Tail on left  | Mean < Median  |

# 2. Boxplot (Box and Whisker Plot)

A **Boxplot** is a graph that summarizes data using **5 important numbers**.

These are called the **Five Number Summary**.

1. Minimum
2. Q1 (First Quartile)
3. Median (Q2)
4. Q3 (Third Quartile)
5. Maximum

---

## Structure

```
Min ──|── [ Q1 | Median | Q3 ] ──|── Max
```

The **box represents the middle 50% of the data**.

---

## Example Data

```
10, 20, 30, 40, 50
```

### Five Number Summary

```
Min = 10
Q1 = 20
Median = 30
Q3 = 40
Max = 50
```

So the box shows the **spread of the middle data**.

---

## Boxplots Are Useful To

* Detect **outliers**
* Compare **multiple datasets**

---

# 3. Scatter Plot

A **Scatter Plot** shows the **relationship between two variables**.

Each point represents **one observation**.

---

## Example

| Study Hours | Marks |
| ----------- | ----- |
| 2           | 40    |
| 3           | 50    |
| 4           | 60    |
| 5           | 70    |

Each pair becomes a **point on the graph**.

---

# Types of Relationships

## 1. Positive Relationship

More study → higher marks

Points go **upward**

---

## 2. Negative Relationship

More speed → less travel time

Points go **downward**

---

## 3. No Relationship

Example:

```
Shoe size vs IQ
```

Points appear **random**.

---

# Very Simple Intuition

| Visualization | Meaning                            |
| ------------- | ---------------------------------- |
| Histogram     | Distribution of one variable       |
| Boxplot       | Summary of distribution            |
| Scatter Plot  | Relationship between two variables |
