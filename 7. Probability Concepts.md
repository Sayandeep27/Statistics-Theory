# Probability Concepts (Easy Explanation with Examples)

Probability is a **mathematical way to measure uncertainty**. It tells us **how likely something is to happen**.

For example:

* What is the chance of getting **Head** when tossing a coin?
* What is the chance of rolling **6** on a dice?

Probability answers such questions.

---

# 1. Random Experiment

A **Random Experiment** is an experiment where the **exact outcome cannot be predicted with certainty**, even though all possible outcomes are known.

### Examples

#### 1. Tossing a coin

Possible results: Head or Tail
But we cannot know beforehand which one will occur.

#### 2. Rolling a dice

Possible results: 1,2,3,4,5,6
But we cannot predict which number will appear.

#### 3. Picking a card from a deck

Possible outcomes: 52 different cards.

So,

**Random Experiment = An experiment with uncertain outcome but known possibilities.**

---

# 2. Sample Space

The **Sample Space** is the **set of all possible outcomes** of a random experiment.

It is usually denoted by **S**.

### Example 1: Tossing a coin

Possible outcomes:

```
S = {H, T}
```

Where

H = Head
T = Tail

---

### Example 2: Rolling a dice

```
S = {1,2,3,4,5,6}
```

---

### Example 3: Tossing two coins

Possible outcomes:

```
S = {HH, HT, TH, TT}
```

So,

**Sample Space = All possible outcomes of an experiment.**

---

# 3. Event

An **Event** is a **subset of the sample space**.

In simple words:

Event = **one or more outcomes we are interested in**

---

### Example: Rolling a dice

Sample Space

```
S = {1,2,3,4,5,6}
```

Event A = Getting an even number

```
A = {2,4,6}
```

Event B = Getting a number greater than 4

```
B = {5,6}
```

---

### Types of Events

#### 1. Simple Event

Only **one outcome**

Example:

```
A = {3}
```

---

#### 2. Compound Event

More than **one outcome**

Example:

```
A = {2,4,6}
```

---

# 4. Components of an Event

The **components of an event** are the **individual outcomes inside that event**.

Example:

```
Event A = {2,4,6}
```

Components of event A:

```
2, 4, 6
```

Each element inside the event is called a **sample point** or **outcome**.

---

# 5. Probability

Probability measures **how likely an event is to occur**.

Probability value always lies between **0 and 1**.

```
0 ≤ P(A) ≤ 1
```

Where:

```
P(A) = Probability of event A
```

---

### Probability Formula

```
P(A) = Number of favorable outcomes / Total number of outcomes
```

---

### Example: Rolling a dice

Event A = getting an even number

Even numbers:

```
{2,4,6}
```

Total outcomes = 6
Favorable outcomes = 3

Probability:

```
P(A) = 3 / 6 = 0.5
```

So there is **50% chance** of getting an even number.

---

# 6. Probability Function

A **Probability Function** assigns a **probability value to each event**.

It must satisfy **three rules**.

---

### Rule 1

Probability cannot be negative.

```
P(A) ≥ 0
```

---

### Rule 2

Probability cannot exceed 1.

```
P(A) ≤ 1
```

---

### Rule 3

Probability of the **entire sample space is 1**.

```
P(S) = 1
```

Meaning:

Something from the sample space **must occur**.

---

# 7. Joint Events

**Joint Events** occur when **two events happen together**.

Mathematically written as:

```
A ∩ B
```

Which means:

**A AND B occur together**

---

### Example

Rolling a dice

Event A = Even number

```
A = {2,4,6}
```

Event B = Number greater than 3

```
B = {4,5,6}
```

Joint event:

```
A ∩ B = {4,6}
```

These outcomes satisfy **both conditions**.

---

### Probability of Joint Events

```
P(A ∩ B)
```

This means **probability that both A and B occur together**.

---

# 8. Disjoint Events (Mutually Exclusive Events)

Two events are **Disjoint** if they **cannot occur at the same time**.

So,

```
A ∩ B = ∅
```

(Empty set)

---

### Example

Rolling a dice

Event A = getting **2**

```
A = {2}
```

Event B = getting **5**

```
B = {5}
```

They cannot happen together in a single roll.

So they are **disjoint events**.

---

# 9. Dependent Events

Two events are **dependent** if **the occurrence of one event affects the probability of the other event**.

---

### Example

Drawing cards **without replacement**

Event A = First card is Ace
Event B = Second card is Ace

If the first card is Ace, the probability of the second Ace **changes**.

So events are **dependent**.

---

# 10. Independent Events

Two events are **independent** if **one event does NOT affect the other**.

---

### Example

Tossing a coin twice.

Event A = First toss is Head
Event B = Second toss is Head

The result of the first toss **does not affect** the second toss.

So they are **independent events**.

---

### Probability Rule for Independent Events

```
P(A ∩ B) = P(A)P(B)
```

---

### Example

Coin toss twice

```
P(H) = 1/2
```

Probability of getting **Head twice**:

```
P(HH) = 1/2 × 1/2 = 1/4
```

---

# 11. Disjoint vs Independent (Very Important)

| Concept            | Meaning                                     |
| ------------------ | ------------------------------------------- |
| Disjoint Events    | Cannot occur together                       |
| Independent Events | Occurrence of one does not affect the other |

Important fact:

**Disjoint events are NEVER independent.**

Because if event A happens, event B **cannot happen**.

---

# 12. Quick Real-Life Example

Suppose a bag has:

* 3 Red balls
* 2 Blue balls

Total = 5 balls

Sample Space:

```
S = {R1,R2,R3,B1,B2}
```

Event A = picking a Red ball

```
A = {R1,R2,R3}
```

Event B = picking a Blue ball

```
B = {B1,B2}
```

Probability of Red:

```
P(A) = 3/5
```

Probability of Blue:

```
P(B) = 2/5
```
