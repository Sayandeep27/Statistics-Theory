# Chi-Square Test Explained (Complete Guide)

---

# 1. What is a Chi-Square Test?

The **Chi-Square Test (χ² test)** is a statistical hypothesis test used to determine whether there is a **significant relationship between categorical variables**.

Unlike Z-tests and T-tests which compare **means**, the Chi-Square test compares **frequencies or counts**.

It answers questions like:

* Is there a relationship between **gender and product preference**?
* Is a **dice fair or biased**?
* Are two categorical variables **independent or related**?

The Chi-square test is widely used in **statistics, machine learning, market research, and feature selection**.

---

# 2. When Do We Use a Chi-Square Test?

A Chi-square test is used when the following conditions are satisfied:

| Condition          | Explanation                                             |
| ------------------ | ------------------------------------------------------- |
| Data type          | Data must be **categorical**                            |
| Observations       | Data must represent **frequencies (counts)**            |
| Independence       | Observations should be independent                      |
| Expected frequency | Expected value in each cell should generally be **≥ 5** |

Example categorical variables:

* Gender (Male / Female)
* Product choice (A / B / C)
* Education level

---

# 3. Chi-Square Test Formula

The Chi-square statistic is calculated using:

```
χ² = Σ (O − E)² / E
```

Where:

| Symbol | Meaning              |
| ------ | -------------------- |
| O      | Observed frequency   |
| E      | Expected frequency   |
| χ²     | Chi-square statistic |

The test measures how much the **observed values differ from expected values**.

---

# 4. Observed vs Expected Frequency

**Observed frequency (O)**

Actual data collected from an experiment.

**Expected frequency (E)**

Values we would expect **if the null hypothesis were true**.

Expected frequency is calculated using:

```
Expected = (Row Total × Column Total) / Grand Total
```

---

# 5. Types of Chi-Square Tests

There are two main types of Chi-square tests.

| Test                            | Purpose                                                       |
| ------------------------------- | ------------------------------------------------------------- |
| Chi-Square Test of Independence | Checks if two categorical variables are related               |
| Chi-Square Goodness-of-Fit Test | Checks if observed distribution matches expected distribution |

---

# 6. Chi-Square Test of Independence (Example)

## Problem

A researcher wants to know whether **gender influences product preference**.

Survey results:

| Gender | Product A | Product B | Total |
| ------ | --------- | --------- | ----- |
| Male   | 30        | 20        | 50    |
| Female | 20        | 30        | 50    |
| Total  | 50        | 50        | 100   |

---

# Step 1 — Define Hypotheses

Null Hypothesis (H0)

```
Gender and product preference are independent.
```

Alternative Hypothesis (H1)

```
Gender and product preference are related.
```

---

# Step 2 — Calculate Expected Frequencies

Formula:

```
Expected = (Row Total × Column Total) / Grand Total
```

Example:

Expected value for Male choosing Product A:

```
E = (50 × 50) / 100
E = 25
```

Expected table:

| Gender | Product A | Product B |
| ------ | --------- | --------- |
| Male   | 25        | 25        |
| Female | 25        | 25        |

---

# Step 3 — Compute Chi-Square Statistic

```
χ² = Σ (O − E)² / E
```

Calculation:

| Cell     | O  | E  | (O−E)²/E |
| -------- | -- | -- | -------- |
| Male A   | 30 | 25 | 1        |
| Male B   | 20 | 25 | 1        |
| Female A | 20 | 25 | 1        |
| Female B | 30 | 25 | 1        |

Total:

```
χ² = 4
```

---

# Step 4 — Calculate Degrees of Freedom

Degrees of freedom:

```
df = (rows − 1) × (columns − 1)
```

Example:

```
df = (2 − 1) × (2 − 1)

df = 1
```

---

# Step 5 — Compare with Critical Value

At significance level:

```
α = 0.05
```

From Chi-square table:

```
Critical value ≈ 3.84
```

Decision rule:

| Condition | Decision          |
| --------- | ----------------- |
| χ² > 3.84 | Reject H0         |
| χ² ≤ 3.84 | Fail to reject H0 |

Our result:

```
χ² = 4
```

Since:

```
4 > 3.84
```

We **reject the null hypothesis**.

---

# 7. Final Conclusion

There is **statistical evidence that gender and product preference are related**.

---

# 8. Chi-Square Goodness-of-Fit Test

This test checks whether **observed data fits a theoretical distribution**.

Example:

Testing whether a **dice is fair**.

Expected counts:

| Outcome | Expected |
| ------- | -------- |
| 1       | 10       |
| 2       | 10       |
| 3       | 10       |
| 4       | 10       |
| 5       | 10       |
| 6       | 10       |

If observed counts are significantly different, the dice may be **biased**.

---

# 9. Chi-Square in Machine Learning

Chi-square tests are widely used in **feature selection** for classification models.

Example:

Determining whether a feature like **"Gender" influences loan approval**.

If the p-value is small, the feature is **important for prediction**.

Commonly used with:

* Logistic regression
* Decision trees
* Naive Bayes

# 10. Python Implementation of Chi-Square Test

```python
import scipy.stats as stats
import numpy as np

# observed data
observed = np.array([[30,20],[20,30]])

chi2_stat, p_value, dof, expected = stats.chi2_contingency(observed)

print("Chi-square statistic:", chi2_stat)
print("P-value:", p_value)
print("Degrees of freedom:", dof)
print("Expected frequencies:\n", expected)
```

| Feature      | Chi-Square              | Z-Test    | T-Test         |
| ------------ | ----------------------- | --------- | -------------- |
| Data type    | Categorical             | Numerical | Numerical      |
| Compares     | Frequencies             | Means     | Means          |
| Distribution | Chi-square distribution | Normal    | t-distribution |
