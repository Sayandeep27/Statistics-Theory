# 1. Z-Test

## What is a Z-Test?

A **Z-Test** is used to determine whether the **mean of a sample is significantly different from the population mean**.

### Conditions for Z-Test

* Sample size **greater than 30**
* Population variance **known**
* Data approximately **normally distributed**

---

## Example Problem

Suppose the **average height of a population is 170 cm**.

We collect a sample of student heights and want to test:

**Is the sample mean significantly different from 170 cm?**

---

## Python Implementation

```python
import numpy as np
from statsmodels.stats.weightstats import ztest

# sample data
heights = [172,168,174,169,171,173,170,175,169,172]

# perform z-test
z_stat, p_value = ztest(heights, value=170)

print("Z Statistic:", z_stat)
print("P-value:", p_value)

if p_value < 0.05:
    print("Reject Null Hypothesis")
else:
    print("Fail to Reject Null Hypothesis")
```

---

## Output Interpretation

| Term        | Meaning                                      |
| ----------- | -------------------------------------------- |
| Z-statistic | Distance of sample mean from population mean |
| p-value     | Probability of observing the result          |

If

```
p-value < 0.05
```

We **reject the null hypothesis**.

# 2. T-Test

## What is a T-Test?

A **T-Test** is used when:

* Sample size **less than 30**
* Population variance **unknown**

It determines whether the **means of groups are statistically different**.

---

## Types of T-Test

| Type               | Purpose                                  |
| ------------------ | ---------------------------------------- |
| One Sample T-Test  | Compare sample mean with population mean |
| Independent T-Test | Compare two independent groups           |
| Paired T-Test      | Compare before-after measurements        |

---

## One Sample T-Test Example

```python
import numpy as np
from scipy import stats

data = [22,21,23,24,25,26,22,23]

t_stat, p_value = stats.ttest_1samp(data, popmean=20)

print("T-statistic:", t_stat)
print("P-value:", p_value)

if p_value < 0.05:
    print("Reject Null Hypothesis")
else:
    print("Fail to Reject Null Hypothesis")
```

---

## Independent T-Test Example

Example: Compare marks of two classes.

```python
import numpy as np
from scipy.stats import ttest_ind

class_A = [80,85,88,90,86]
class_B = [78,82,85,87,83]

t_stat, p_value = ttest_ind(class_A, class_B)

print("T-statistic:", t_stat)
print("P-value:", p_value)
```

# 3. Chi-Square Test

## What is a Chi-Square Test?

A **Chi-Square Test** checks the **relationship between two categorical variables**.

It is widely used for:

* Feature selection
* Association testing
* Categorical data analysis

---

## Example Problem

We want to check whether **gender affects product purchase behavior**.

---

## Contingency Table

| Gender | Purchased | Not Purchased |
| ------ | --------- | ------------- |
| Male   | 60        | 40            |
| Female | 30        | 70            |

---

## Python Implementation

```python
import numpy as np
from scipy.stats import chi2_contingency

# contingency table
purchase_data = np.array([[60,40],
                          [30,70]])

chi2, p, dof, expected = chi2_contingency(purchase_data)

print("Chi-Square Statistic:", chi2)
print("P-value:", p)
print("Degrees of Freedom:", dof)
print("Expected Frequencies:\n", expected)

if p < 0.05:
    print("Reject Null Hypothesis (Variables are dependent)")
else:
    print("Fail to Reject Null Hypothesis (Variables are independent)")
```

---

## Output Explanation

| Output               | Meaning                                         |
| -------------------- | ----------------------------------------------- |
| Chi-square statistic | Difference between observed and expected values |
| p-value              | Probability of observing such difference        |
| Degrees of freedom   | Independent values in table                     |
| Expected frequencies | Expected counts if variables are independent    |

# 4. ANOVA (Analysis of Variance)

## What is ANOVA?

**ANOVA** is used to determine whether **three or more group means are statistically different**.

Instead of comparing means pairwise, ANOVA checks them **all at once**.

---

## Example Problem

Compare the **average marks of students in three classes**.

---

## Python Implementation

```python
from scipy.stats import f_oneway

class_A = [85,88,90,87,86]
class_B = [78,82,85,80,79]
class_C = [92,94,91,95,93]

f_stat, p_value = f_oneway(class_A, class_B, class_C)

print("F Statistic:", f_stat)
print("P-value:", p_value)

if p_value < 0.05:
    print("Reject Null Hypothesis (At least one mean is different)")
else:
    print("Fail to Reject Null Hypothesis")
```

---

## Interpretation

| Term        | Meaning                                                  |
| ----------- | -------------------------------------------------------- |
| F-statistic | Ratio of between-group variance to within-group variance |
| p-value     | Probability that group means are equal                   |

If

```
p-value < 0.05
```

At least **one group mean is significantly different**.
