# T-Test Explained (Complete Guide)

---

# 1. What is a T-Test?

A **T-test** is a statistical hypothesis test used to determine whether the **mean of a sample is significantly different from a population mean or another sample mean**.

It is mainly used when:

* The **sample size is small (n < 30)**
* The **population standard deviation is unknown**

Instead of the normal distribution used in Z-tests, the **t-distribution** is used.

T-tests are widely used in **statistics, machine learning experiments, scientific research, and A/B testing**.

---

# 2. When Do We Use a T-Test?

A T-test is used when the following conditions are satisfied:

| Condition                             | Explanation                              |
| ------------------------------------- | ---------------------------------------- |
| Small sample size                     | Sample size is **less than 30**          |
| Population standard deviation unknown | σ is not known                           |
| Data approximately normal             | Data roughly follows normal distribution |

If the **sample size is large and population standard deviation is known**, a **Z-test** is usually used instead.

---

# 3. T-Test Formula

The t statistic is calculated using the following formula:

```
t = (x̄ − μ) / (s / √n)
```

Where:

| Symbol | Meaning                   |
| ------ | ------------------------- |
| x̄     | Sample mean               |
| μ      | Population mean           |
| s      | Sample standard deviation |
| n      | Sample size               |
| t      | t statistic               |

The **t value measures how far the sample mean is from the population mean in terms of standard error**.

---

# 4. Understanding the t-Distribution

The **t-distribution** looks similar to the normal distribution but has **heavier tails**.

This happens because we are estimating the standard deviation from the sample.

As the **sample size increases**, the t-distribution becomes **closer to the normal distribution**.

---

# 5. Degrees of Freedom

An important concept in the t-test is **degrees of freedom (df)**.

```
df = n − 1
```

Where:

* n = sample size

Degrees of freedom determine **which t-distribution curve is used**.

Example:

If sample size = 10

```
df = 10 − 1 = 9
```

# 6. Step-by-Step Example of a T-Test

## Problem

A teacher claims that the **average score of students is 70 marks**.

A random sample of **10 students** has the following scores:

```
65, 70, 68, 72, 66, 75, 69, 71, 67, 73
```

We want to test whether the **average score is different from 70**.

---

# Step 1 — Define Hypotheses

**Null Hypothesis (H0)**

```
H0: μ = 70
```

**Alternative Hypothesis (H1)**

```
H1: μ ≠ 70
```

This is a **two-tailed test**.

---

# Step 2 — Calculate Sample Mean

```
Sample mean (x̄) = 69.6
```

---

# Step 3 — Calculate Sample Standard Deviation

Assume:

```
s = 3.2
```

---

# Step 4 — Apply the Formula

```
t = (69.6 − 70) / (3.2 / √10)
```

```
t = −0.4 / 1.01
```

```
t = −0.396
```

---

# Step 5 — Determine Critical Value

Significance level:

```
α = 0.05
```

Degrees of freedom:

```
df = n − 1 = 9
```

From t-table:

```
Critical value ≈ ±2.262
```

---

# Step 6 — Decision Rule

| Condition | Decision |         |                   |
| --------- | -------- | ------- | ----------------- |
|           | t        | > 2.262 | Reject H0         |
|           | t        | ≤ 2.262 | Fail to reject H0 |

Calculated value:

```
t = −0.396
```

Since:

```
|−0.396| < 2.262
```

We **fail to reject the null hypothesis**.

---

# 7. Final Conclusion

There is **not enough statistical evidence** to conclude that the average student score is different from 70.

The difference observed could be due to **random variation in the sample**.

# 8. Types of T-Tests

There are **three main types of T-tests**.

| Type               | Purpose                                                   |
| ------------------ | --------------------------------------------------------- |
| One-sample t-test  | Compare sample mean with population mean                  |
| Independent t-test | Compare means of two independent groups                   |
| Paired t-test      | Compare means of the same group before and after a change |

---

# 9. One-Sample T-Test

Used when comparing **one sample mean with a population mean**.

Example:

Testing whether the **average weight of apples from a farm equals 200g**.

---

# 10. Independent T-Test

Used when comparing **means of two different groups**.

Example:

Comparing:

* Exam scores of **students taught by two different teachers**

---

# 11. Paired T-Test

Used when comparing **the same group before and after an event**.

Example:

Testing whether a **training program improved employee productivity**.

---

# 12. One-Tailed vs Two-Tailed Tests

Understanding **one-tailed and two-tailed tests** is essential in hypothesis testing.

---

# One-Tailed Test

A **one-tailed test** checks whether a parameter is either:

* Greater than a value
* Less than a value

Example:

```
H0: μ = 50
H1: μ > 50
```

Right-tailed test.

Another example:

```
H0: μ = 50
H1: μ < 50
```

Left-tailed test.

---

# Two-Tailed Test

A **two-tailed test** checks whether a parameter is **different from a value**.

Example:

```
H0: μ = 50
H1: μ ≠ 50
```

This means we check **both directions of the distribution**.

---

# 13. How to Know When to Use One-Tailed or Two-Tailed

Use the following rule:

| Situation                            | Test Type  |
| ------------------------------------ | ---------- |
| Testing if value is greater          | One-tailed |
| Testing if value is smaller          | One-tailed |
| Testing if value is simply different | Two-tailed |

---

# Example Intuition

| Question                                            | Test       |
| --------------------------------------------------- | ---------- |
| Is a new drug **more effective**?                   | One-tailed |
| Is a defect rate **lower** than before?             | One-tailed |
| Is the average salary **different** from last year? | Two-tailed |

# 15. Z-Test vs T-Test

| Feature                       | Z-Test      | T-Test         |
| ----------------------------- | ----------- | -------------- |
| Sample size                   | Large (≥30) | Small (<30)    |
| Population standard deviation | Known       | Unknown        |
| Distribution                  | Normal      | t-distribution |

# 16. Python Implementation of T-Test

```python
import numpy as np
from scipy import stats

sample_data = [65,70,68,72,66,75,69,71,67,73]

population_mean = 70

# one sample t-test

t_statistic, p_value = stats.ttest_1samp(sample_data, population_mean)

print("T statistic:", t_statistic)
print("P value:", p_value)
```
