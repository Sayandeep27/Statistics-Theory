# Covariance and Correlation in Statistics and Machine Learning

---

## 1. Introduction

In statistics and machine learning, understanding relationships between variables is extremely important. Many real-world problems require us to understand whether two variables move together and how strongly they are related.

Examples:

| Variable 1    | Variable 2      |
| ------------- | --------------- |
| Hours studied | Exam marks      |
| Temperature   | Ice-cream sales |
| Salary        | Spending        |
| Exercise time | Weight          |

Important questions:

* If **one variable increases**, does the **other also increase**?
* If **one increases**, does the **other decrease**?
* Or are they **not related at all**?

To measure these relationships we use two key statistical concepts:

1. **Covariance**
2. **Correlation**

Correlation is essentially a **standardized and improved version of covariance**.

---

# 2. Covariance

## Definition

**Covariance measures how two variables change together.**

It tells us whether two variables move in the **same direction** or **opposite directions**.

---

## Simple Intuition

Suppose we observe the following dataset:

| Hours Studied | Marks |
| ------------- | ----- |
| 2             | 50    |
| 4             | 60    |
| 6             | 70    |
| 8             | 80    |

Observation:

When **study hours increase**, **marks also increase**.

This means the variables **move together**, which results in **positive covariance**.

---

## Case 1: Positive Covariance

Both variables move in the **same direction**.

Example:

| X  | Y  |
| -- | -- |
| 10 | 20 |
| 20 | 30 |
| 30 | 40 |

When **X increases → Y increases**.

Therefore:

**Covariance > 0**

Real examples:

* Study time and marks
* Advertising and sales
* Experience and salary

---

## Case 2: Negative Covariance

Variables move in **opposite directions**.

Example:

| X  | Y  |
| -- | -- |
| 10 | 90 |
| 20 | 70 |
| 30 | 50 |

When **X increases → Y decreases**.

Therefore:

**Covariance < 0**

Examples:

* Price and demand
* Speed and travel time

---

## Case 3: Zero Covariance

If there is **no clear relationship**, covariance is close to zero.

Example:

| X  | Y  |
| -- | -- |
| 10 | 50 |
| 20 | 10 |
| 30 | 70 |

No pattern exists between the variables.

Therefore:

**Covariance ≈ 0**

---

# 3. Covariance Formula

Population covariance formula:

```
Cov(X,Y) = Σ (Xi − X̄)(Yi − Ȳ) / N
```

Where:

| Symbol | Meaning                |
| ------ | ---------------------- |
| Xi     | Value of variable X    |
| Yi     | Value of variable Y    |
| X̄     | Mean of X              |
| Ȳ      | Mean of Y              |
| N      | Number of observations |

For **sample covariance**, the denominator becomes **n − 1**.

---

## Intuition Behind the Formula

The formula performs the following steps:

1. Compute the **mean of X and Y**.
2. Calculate how far each value is from its mean.
3. Multiply the deviations.
4. Average the results.

If both deviations have the **same sign**, the product is positive.

If the signs are **different**, the product becomes negative.

---

### Why Signs Matter

| Xi   | Yi   | Xi − X̄ | Yi − Ȳ | Product |
| ---- | ---- | ------- | ------ | ------- |
| High | High | +       | +      | +       |
| Low  | Low  | −       | −      | +       |

When variables move together → **positive covariance**.

Opposite movement:

| Xi   | Yi   | Xi − X̄ | Yi − Ȳ | Product |
| ---- | ---- | ------- | ------ | ------- |
| High | Low  | +       | −      | −       |
| Low  | High | −       | +      | −       |

Result → **negative covariance**.

---

# 4. Limitation of Covariance

Covariance indicates **direction of relationship**, but it has an important limitation.

It does **not clearly indicate the strength of the relationship**.

This is because covariance **depends on measurement units**.

Example:

* Height measured in meters
* Height measured in centimeters

Changing units changes the covariance value.

Therefore it becomes difficult to compare relationships.

To solve this issue we use **Correlation**.

---

# 5. Correlation

## Definition

**Correlation measures both the direction and strength of the relationship between two variables.**

It is a **normalized version of covariance**.

---

## Correlation Formula

```
r = Cov(X,Y) / (σX σY)
```

Where:

| Symbol   | Meaning                 |
| -------- | ----------------------- |
| Cov(X,Y) | Covariance              |
| σX       | Standard deviation of X |
| σY       | Standard deviation of Y |
| r        | Correlation coefficient |

---

# 6. Range of Correlation

Correlation always lies within the following range:

```
-1 ≤ r ≤ 1
```

Interpretation:

| Correlation Value | Meaning                        |
| ----------------- | ------------------------------ |
| +1                | Perfect positive relationship  |
| 0.8               | Strong positive relationship   |
| 0.5               | Moderate relationship          |
| 0                 | No relationship                |
| -0.5              | Moderate negative relationship |
| -1                | Perfect negative relationship  |

---

# 7. Visual Understanding

Example: Study time vs marks.


### Zero Correlation

# 8. Covariance vs Correlation

| Feature                | Covariance | Correlation |
| ---------------------- | ---------- | ----------- |
| Measures relationship  | Yes        | Yes         |
| Shows direction        | Yes        | Yes         |
| Shows strength clearly | No         | Yes         |
| Range                  | −∞ to +∞   | −1 to +1    |
| Unit dependent         | Yes        | No          |
| Easy to interpret      | No         | Yes         |

---

# 9. Real-Life Example

Suppose we analyze the relationship between **advertising spending and sales**.

| Advertising Spend | Sales |
| ----------------- | ----- |
| 10                | 100   |
| 20                | 150   |
| 30                | 200   |
| 40                | 260   |

Observation:

More advertising → higher sales.

Therefore:

* Covariance is **positive**
* Correlation is **close to +1**

---

# 10. Covariance and Correlation in Python

```python
import numpy as np

x = [2,4,6,8]
y = [50,60,70,80]

# Covariance
cov = np.cov(x, y)

# Correlation
corr = np.corrcoef(x, y)

print("Covariance matrix:\n", cov)
print("Correlation matrix:\n", corr)
```

# 11. How Correlation is Used in Machine Learning

Correlation plays a major role in **Exploratory Data Analysis (EDA)** and **Feature Engineering**.

It helps identify **which features are important for predicting the target variable**.

---

## Feature Selection

Example dataset:

| Study Hours | Sleep Hours | TV Time | Marks |
| ----------- | ----------- | ------- | ----- |
| 2           | 7           | 3       | 50    |
| 4           | 6           | 2       | 60    |
| 6           | 6           | 1       | 70    |
| 8           | 5           | 1       | 85    |

Correlation with target variable (Marks):

| Feature     | Correlation |
| ----------- | ----------- |
| Study Hours | 0.92        |
| Sleep Hours | 0.20        |
| TV Time     | -0.75       |

Interpretation:

* **Study Hours** strongly increases marks
* **TV Time** strongly decreases marks
* **Sleep Hours** has weak impact

Therefore models should mainly learn from:

* Study Hours
* TV Time

---

## Removing Redundant Features (Multicollinearity)

Sometimes features are strongly correlated with each other.

Example:

| Feature     | Description     |
| ----------- | --------------- |
| Engine Size | Engine capacity |
| Horsepower  | Engine power    |

If correlation between them is **0.95**, they provide almost the same information.

This creates **multicollinearity**, which negatively affects models such as:

* Linear Regression
* Logistic Regression

Solution:

Remove one of the correlated variables.

---

## Correlation Matrix

A correlation matrix shows relationships between all variables.

Example:

| Feature    | Age  | Salary | Experience |
| ---------- | ---- | ------ | ---------- |
| Age        | 1    | 0.65   | 0.80       |
| Salary     | 0.65 | 1      | 0.75       |
| Experience | 0.80 | 0.75   | 1          |

---

### Python Example

```python
import pandas as pd

correlation_matrix = df.corr()
print(correlation_matrix)
```

Visualization:

```python
import seaborn as sns
import matplotlib.pyplot as plt

sns.heatmap(df.corr(), annot=True)
plt.show()
```

This produces a **correlation heatmap** which visually shows strong and weak relationships.

---

## Detecting Important Variables

Example: Predicting house prices.

| Feature            | Correlation with Price |
| ------------------ | ---------------------- |
| Area               | 0.85                   |
| Bedrooms           | 0.70                   |
| Distance from city | -0.60                  |
| Paint color        | 0.02                   |

Observation:

Paint color has almost **no correlation** with price.

Therefore it can be **removed from the model**.

---

## Linear Regression and Correlation

Linear regression models relationships between variables using the equation:

```
y = mx + b
```

Where:

| Symbol | Meaning         |
| ------ | --------------- |
| y      | Target variable |
| x      | Feature         |
| m      | Slope           |
| b      | Intercept       |

If correlation between **x and y** is strong, the regression model performs better.

# 12. Why Correlation Does NOT Imply Causation

One of the most important principles in statistics is:

**Correlation does NOT mean causation.**

Just because two variables move together does not mean one causes the other.

---

## Example: Ice Cream Sales and Drowning

Observation:

| Ice Cream Sales | Drowning Cases |
| --------------- | -------------- |
| Low             | Low            |
| High            | High           |

There appears to be a positive correlation.

But eating ice cream does **not cause drowning**.

The real cause is a hidden variable:

**Hot weather (summer)**.

Hot weather increases both:

* Ice cream consumption
* Swimming activity

Which indirectly increases drowning incidents.

This hidden variable is called a **confounding variable**.

---

## Example: Shoe Size vs Reading Ability

Observation in children:

| Shoe Size | Reading Ability |
| --------- | --------------- |
| Small     | Low             |
| Large     | High            |

Correlation exists, but bigger shoes do **not cause better reading ability**.

The real factor is:

**Age**.

Older children:

* Have larger shoe sizes
* Have better reading skills

---

## Spurious Correlations

Some correlations are completely coincidental.

A famous example showed correlation between:

* Number of films **Nicolas Cage** appeared in
* Number of **swimming pool drownings**

These variables have no real relationship. They are simply **spurious correlations**.

---

# 13. Why This Matters in Machine Learning

Machine learning models learn **patterns**, not **true causes**.

If a dataset contains misleading correlations, the model may learn incorrect relationships.

Example dataset for loan approval:

| Feature     | Correlation with Loan Approval |
| ----------- | ------------------------------ |
| Salary      | 0.80                           |
| Age         | 0.60                           |
| Postal Code | 0.75                           |

Postal code might correlate with approval due to economic location differences, but postal code itself does **not cause loan approval**.

Blindly relying on correlation can create **biased or misleading models**.

---

# 14. Best Practices in Machine Learning

To properly use correlation in machine learning:

1. Use correlation during **exploratory data analysis**.
2. Apply **domain knowledge**.
3. Look for possible **confounding variables**.
4. Validate results using **experiments or validation datasets**.

* **Covariance** tells us the direction of relationship.
* **Correlation** tells us both direction and strength.
* In machine learning, correlation is widely used for **feature analysis and data understanding**, but it must be interpreted carefully because **correlation alone does not prove causation**.
