# 1. Bernoulli Distribution

## Idea

A **Bernoulli distribution** represents an experiment that has **only two possible outcomes**.

Example outcomes:

* Success / Failure
* Yes / No
* 1 / 0
* True / False

So every trial produces **one of two results**.

---

## Example

Suppose you toss a coin once.

Possible outcomes:

* Head → Success (1)
* Tail → Failure (0)

If the probability of success is **p**, then the probability of failure is:

```
1 - p
```

---

## Probability Function

For a Bernoulli random variable **X**:

```
P(X=x) = p^x (1-p)^{1-x}
```

Where:

| Symbol | Meaning                |
| ------ | ---------------------- |
| x      | Outcome (0 or 1)       |
| p      | Probability of success |
| 1-p    | Probability of failure |

---

## Example Calculation

Suppose

Probability of passing an exam = **0.7**

Then:

| Outcome  | Probability |
| -------- | ----------- |
| Pass (1) | 0.7         |
| Fail (0) | 0.3         |

This is a **Bernoulli distribution**.

---

## Where Bernoulli is used

* Email spam detection (Spam / Not spam)
* Medical test (Positive / Negative)
* Ad click prediction (Clicked / Not clicked)

# 2. Binomial Distribution

## Idea

A **Binomial distribution** describes the number of successes in **multiple Bernoulli trials**.

So:

Bernoulli = **one trial**

Binomial = **many Bernoulli trials**

---

## Conditions for Binomial Distribution

Four conditions must hold:

1. Fixed number of trials **n**
2. Only **two outcomes**
3. Probability **p remains constant**
4. Trials are **independent**

---

## Example

Suppose you toss a coin **5 times**.

Question:

What is the probability of getting **exactly 3 heads**?

---

## Formula

```
P(X=k) =
(n choose k)
p^k
(1-p)^{n-k}
```

Where:

| Symbol | Meaning                |
| ------ | ---------------------- |
| n      | number of trials       |
| k      | number of successes    |
| p      | probability of success |

---

## Example Calculation

Coin toss example:

n = 5
k = 3
p = 0.5

```
P(X=3) =
(5 choose 3)
(0.5)^3
(0.5)^2
```

```
= 10 × 0.125 × 0.25
```

```
= 0.3125
```

So probability of getting **3 heads in 5 tosses = 0.3125**

---

## Real Life Examples

* Number of customers who buy a product
* Number of students who pass
* Number of defective items in a sample

# 3. Continuous Distribution

Discrete distributions deal with **countable values**.

Example:

0,1,2,3

Continuous distributions deal with **any value within a range**.

Example:

Height = 170.4 cm
Weight = 65.82 kg
Temperature = 27.35°C

These can take **infinite values**.

Two important continuous distributions:

1. Uniform distribution
2. Normal distribution

---

# 3.1 Uniform Distribution

## Idea

In a **uniform distribution**, every value in a range has **equal probability**.

---

## Example

Suppose a bus arrives between **10:00 and 10:30**.

Arrival time is **random**.

So any time between:

10:00 → 10:30

is equally likely.

---

## Probability Graph

Uniform distribution looks like a **flat rectangle**.

All values have equal probability.

```
Probability
   |
   |  ________
   | |        |
   | |        |
   | |        |
   | |________|
   |
   +------------------
       a        b
```

Where:

* **a = minimum value**
* **b = maximum value**

---

## Example

Random number between **0 and 10**

Probability of any value inside the range is equal.

---

# 3.2 Normal Distribution (Gaussian Distribution)

## Idea

The **normal distribution** is the most important distribution in statistics.

It describes many **natural phenomena**.

Examples:

* Human height
* Exam marks
* Measurement errors
* IQ scores

---

## Shape

Normal distribution looks like a **bell curve**.

```
            /\
           /  \
          /    \
         /      \
        /        \
-------/----------\-------
```

Most values are near the **mean (average)**.

---

## Properties

Important characteristics:

1. Bell-shaped curve
2. Symmetrical around mean
3. Mean = Median = Mode
4. Defined by **μ (mean)** and **σ (standard deviation)**

---

## Example

Suppose exam marks follow normal distribution:

Mean = 70
Standard deviation = 10

Most students will score near **70**.

Few will score:

* very low
* very high

---

## Empirical Rule

In normal distribution:

| Range  | Percentage of Data |
| ------ | ------------------ |
| μ ± 1σ | 68%                |
| μ ± 2σ | 95%                |
| μ ± 3σ | 99.7%              |

---

# 4. Poisson Distribution

## Idea

A **Poisson distribution** models the number of times an event occurs in a **fixed time or space interval**.

It is used when events happen **randomly but with a known average rate**.

---

## Examples

* Number of customers arriving per hour
* Number of accidents per day
* Number of emails received per minute
* Number of calls in a call center

---

## Formula

```
P(X=k) =
(e^-λ λ^k) / k!
```

Where:

| Symbol | Meaning                  |
| ------ | ------------------------ |
| λ      | average number of events |
| k      | actual number of events  |
| e      | Euler constant (≈2.718)  |

---

## Example

Suppose:

Average calls per minute = **4**

What is probability of **exactly 2 calls** in a minute?

```
P(X=2) =
(e^-4 4^2) / 2!
```

```
= (e^-4 × 16) / 2
```

```
= 0.1465
```

So probability ≈ **0.1465**

# Very Important Intuition

Think of them like this:

Bernoulli → **One yes/no experiment**

Binomial → **Many yes/no experiments**

Poisson → **Counting events in time**

Uniform → **All values equally likely**

Normal → **Natural measurements (bell curve)**
