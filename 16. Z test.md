# Z-Test Explained (Complete Guide)

---

# 1. What is a Z-Test?

A **Z-test** is a statistical hypothesis test used to determine whether the **mean of a sample is significantly different from a known population mean**.

It helps us decide whether an observed difference happened because of:

* Random chance
* A real statistical effect

Z-tests are widely used in **statistics, machine learning experiments, A/B testing, and quality control**.

---

# 2. When Do We Use a Z-Test?

A Z-test is used when the following conditions are satisfied:

| Condition                           | Explanation                                         |
| ----------------------------------- | --------------------------------------------------- |
| Large sample size                   | Sample size is **greater than or equal to 30**      |
| Population standard deviation known | The true population standard deviation (σ) is known |
| Normal distribution                 | Data is approximately normally distributed          |

If these conditions are not satisfied, we usually use a **T-test instead**.

---

# 3. Z-Test Formula

The Z statistic is calculated using the following formula:

```
Z = (x̄ − μ) / (σ / √n)
```

Where:

| Symbol | Meaning                       |
| ------ | ----------------------------- |
| x̄     | Sample mean                   |
| μ      | Population mean               |
| σ      | Population standard deviation |
| n      | Sample size                   |
| Z      | Z-score (test statistic)      |

The **Z-score tells how many standard deviations the sample mean is away from the population mean**.

---

# 4. Understanding the Z-Score

The Z-score represents **distance from the population mean in terms of standard deviations**.

| Z Score | Interpretation                                  |
| ------- | ----------------------------------------------- |
| 0       | Sample mean equals population mean              |
| 1       | Sample mean is 1 standard deviation above mean  |
| -1      | Sample mean is 1 standard deviation below mean  |
| 2       | Sample mean is 2 standard deviations above mean |
| -2      | Sample mean is 2 standard deviations below mean |

Large absolute Z values indicate a **significant difference**.

---

# 5. Step-by-Step Example of a Z-Test

## Problem

A phone company claims that the **average battery life of a phone is 10 hours**.

We test **40 phones** and observe:

| Parameter                     | Value     |
| ----------------------------- | --------- |
| Sample mean                   | 9.5 hours |
| Population standard deviation | 2 hours   |
| Sample size                   | 40        |
| Claimed mean                  | 10 hours  |

We want to test whether the battery life is **less than 10 hours**.

---

# Step 1 — Define Hypotheses

**Null Hypothesis (H0)**

```
H0: μ = 10
```

Battery life equals 10 hours.

**Alternative Hypothesis (H1)**

```
H1: μ < 10
```

Battery life is less than 10 hours.

---

# Step 2 — Apply the Formula

```
Z = (9.5 − 10) / (2 / √40)
```

```
Z = −0.5 / 0.316
```

```
Z = −1.58
```

---

# Step 3 — Compare With Critical Value

At **5% significance level (α = 0.05)**:

| Test Type       | Critical Z Value |
| --------------- | ---------------- |
| One‑tailed test | -1.645           |
| Two‑tailed test | ±1.96            |

Our calculated value:

```
Z = -1.58
```

Decision rule:

| Condition  | Decision          |
| ---------- | ----------------- |
| Z < -1.645 | Reject H0         |
| Z > -1.645 | Fail to reject H0 |

Since:

```
-1.58 > -1.645
```

We **fail to reject the null hypothesis**.

---

# 6. Final Conclusion

There is **not enough statistical evidence** to conclude that the battery life is less than 10 hours.

The observed difference could be due to **random sampling variation**.

# 7. One-Tailed vs Two-Tailed Tests

Understanding **one-tailed and two-tailed tests** is extremely important in hypothesis testing.

---

# One-Tailed Test

A **one-tailed test** checks whether a parameter is either:

* Greater than a value
* Less than a value

The rejection region exists **only on one side of the distribution**.

### Example

Testing whether a machine produces **less than the expected weight**.

```
H0: μ = 50
H1: μ < 50
```

This is a **left‑tailed test**.

Another example:

```
H0: μ = 50
H1: μ > 50
```

This is a **right‑tailed test**.

---

# Two-Tailed Test

A **two-tailed test** checks whether a parameter is **different from a value**, without specifying the direction.

Rejection regions exist **on both sides of the distribution**.

### Example

```
H0: μ = 50
H1: μ ≠ 50
```

Here we only want to know whether the mean is **different**, not whether it is specifically greater or smaller.

---

# 8. How to Know When to Use One-Tailed or Two-Tailed

Use the following rule:

| Situation                            | Test Type  |
| ------------------------------------ | ---------- |
| Testing if value is greater          | One‑tailed |
| Testing if value is smaller          | One‑tailed |
| Testing if value is simply different | Two‑tailed |

---

# Simple Intuition

| Question Asked                                      | Test Type  |
| --------------------------------------------------- | ---------- |
| Is the new model **better** than the old model?     | One‑tailed |
| Is the defect rate **lower** than before?           | One‑tailed |
| Is the average salary **different** from last year? | Two‑tailed |

---

# 9. Types of Z-Tests

There are three common types of Z-tests.

| Type              | Purpose                                  |
| ----------------- | ---------------------------------------- |
| One-sample Z-test | Compare sample mean with population mean |
| Two-sample Z-test | Compare means of two populations         |
| Proportion Z-test | Compare proportions                      |

---

# 10. Example of Proportion Z-Test

Suppose a website claims that **10% of visitors click an ad**.

We test **500 visitors** and observe **65 clicks**.

We test whether the click rate is actually different.

This is solved using a **proportion Z-test**.

# 12. Z-Test vs T-Test

| Feature                       | Z-Test              | T-Test         |
| ----------------------------- | ------------------- | -------------- |
| Sample size                   | Large (≥30)         | Small (<30)    |
| Population standard deviation | Known               | Unknown        |
| Distribution used             | Normal distribution | t-distribution |

---

# 13. Python Implementation of Z-Test

```python
import numpy as np
from scipy import stats

sample_mean = 9.5
population_mean = 10
population_std = 2
sample_size = 40

z_score = (sample_mean - population_mean) / (population_std / np.sqrt(sample_size))

p_value = stats.norm.cdf(z_score)

print("Z score:", z_score)
print("P value:", p_value)
```
