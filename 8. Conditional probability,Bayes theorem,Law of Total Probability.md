# Probability Concepts: Conditional Probability, Bayes’ Theorem, and Law of Total Probability

---

## 1. Conditional Probability

### Idea

**Conditional probability means the probability of an event occurring given that another event has already happened.**

In simple words:

> We update probability because we already know some information.

Example idea:

* Probability that a person is a **student**
* But if we already know the person is **female**, the probability may change.

---

### Mathematical Definition

```
P(A | B) = P(A ∩ B) / P(B)
```

Where:

| Symbol   | Meaning                              |
| -------- | ------------------------------------ |
| P(A | B) | Probability of A given B             |
| P(A ∩ B) | Probability that A and B both happen |
| P(B)     | Probability that B happens           |

---

### Simple Example (Cards)

Suppose we draw **one card from a deck of 52 cards**.

Events:

* A = card is **King**
* B = card is **a Face Card** (J, Q, K)

#### Step 1: Find probabilities

Face cards = 12
Kings = 4

Probability that card is **King AND Face card**

Since every king is a face card:

```
P(A ∩ B) = 4 / 52
```

Probability of B:

```
P(B) = 12 / 52
```

---

#### Step 2: Apply formula

```
P(A|B) = (4/52) / (12/52)
```

```
P(A|B) = 4 / 12
```

```
P(A|B) = 1 / 3
```

**Result:**

If the card is already known to be a **face card**, the probability it is a **king = 1/3**.

# 2. Bayes’ Theorem

## Idea

**Bayes' Theorem allows us to reverse conditional probability.**

It helps answer:

> If we observe some evidence, what is the probability of the cause?

Very important in:

* Machine Learning
* Medical diagnosis
* Spam filtering
* Fraud detection

---

## Bayes’ Theorem Formula

```
P(A | B) = ( P(B | A) * P(A) ) / P(B)
```

Where:

| Term | Meaning           |                       |
| ---- | ----------------- | --------------------- |
| P(A) | Prior probability |                       |
| P(B  | A)                | Likelihood            |
| P(B) | Evidence          |                       |
| P(A  | B)                | Posterior probability |

---

## Medical Example

Suppose a disease test:

Disease prevalence:

```
P(Disease) = 0.01
```

Test accuracy:

```
P(Positive | Disease) = 0.99
```

False positive:

```
P(Positive | No disease) = 0.05
```

---

### Step 1: Define events

A = person has disease
B = test is positive

We want:

```
P(Disease | Positive)
```

---

### Step 2: Use Bayes theorem

```
P(D|+) = ( P(+|D) * P(D) ) / P(+)
```

But we need **P(+)**.

---

### Step 3: Use Law of Total Probability

```
P(+) = P(+|D)P(D) + P(+|NoD)P(NoD)
```

Substitute values:

```
P(+) = (0.99)(0.01) + (0.05)(0.99)
```

```
P(+) = 0.0099 + 0.0495
```

```
P(+) = 0.0594
```

---

### Step 4: Final probability

```
P(D|+) = (0.99 × 0.01) / 0.0594
```

```
P(D|+) ≈ 0.1667
```

### Result

Even with a **positive test**, probability of disease ≈ **16.7%**.

This surprises many people.

This phenomenon is called the **base rate effect**.

# 3. Law of Total Probability

## Idea

Sometimes an event can happen through **multiple different paths**.

The **Law of Total Probability** tells us:

> The total probability of an event equals the sum of probabilities through all possible conditions.

---

## Formula

If events B₁, B₂, B₃ ... Bₙ form a **partition of the sample space**, then:

```
P(A) =
P(A|B1)P(B1) +
P(A|B2)P(B2) +
...
P(A|Bn)P(Bn)
```

---

## Example (Factory Machines)

A factory has **3 machines**.

| Machine | Production | Defect Rate |
| ------- | ---------- | ----------- |
| M1      | 30%        | 1%          |
| M2      | 50%        | 2%          |
| M3      | 20%        | 3%          |

We want the **probability that a random product is defective**.

---

### Step 1: Apply law

```
P(Defect) =
P(Defect|M1)P(M1) +
P(Defect|M2)P(M2) +
P(Defect|M3)P(M3)
```

---

### Step 2: Substitute values

```
P(D) = (0.01)(0.30) + (0.02)(0.50) + (0.03)(0.20)
```

```
P(D) = 0.003 + 0.01 + 0.006
```

```
P(D) = 0.019
```

### Result

Probability of defect = **1.9%**.

---

# Relationship Between the Three

These three concepts are **strongly connected**.

| Concept                  | Purpose                            |
| ------------------------ | ---------------------------------- |
| Conditional Probability  | Probability with extra information |
| Law of Total Probability | Calculates overall probability     |
| Bayes’ Theorem           | Reverses conditional probability   |

---

## Flow of Logic

```
Conditional Probability
        ↓
Law of Total Probability
        ↓
Bayes' Theorem
```

Bayes theorem is actually derived from **conditional probability**.
