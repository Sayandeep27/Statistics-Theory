# Inferential Statistics — Complete Guide

---

# 1. Inferential Statistics

## Definition

**Inferential statistics** is a branch of statistics used to **make conclusions about a large population using a small sample of data**.

Instead of collecting data from **every individual in the population**, we collect data from a **small subset called a sample** and use statistical techniques to **infer or estimate characteristics of the entire population**.

---

## Example

Suppose a city has:

* **1,000,000 voters**

It is difficult to ask every voter about their preference.

So researchers survey:

* **1,000 voters (sample)**

Using the responses from the sample, they estimate:

* Which candidate will win
* Percentage of voters supporting each candidate

This process of using **sample data to draw conclusions about the population** is called **inferential statistics**.

---

# 2. Point Estimation

## Definition

**Point estimation** means **estimating a population parameter using a single numerical value derived from sample data**.

A single value that estimates the population parameter is called a **point estimate**.

---

## Example

A researcher wants to estimate the **average height of students in a college**.

Steps:

1. Select a sample of **50 students**
2. Calculate their average height

Suppose the sample mean is:

```
170 cm
```

The researcher estimates:

```
Average height of all students ≈ 170 cm
```

Here:

| Term             | Meaning          |
| ---------------- | ---------------- |
| Population value | Unknown          |
| Sample value     | Used as estimate |
| Point estimate   | 170 cm           |

---

# 3. Interval Estimation

## Definition

**Interval estimation** provides **a range of values within which the true population parameter is likely to lie**.

Instead of giving a single estimate, statistics provides **a range that likely contains the true value**.

---

## Example

Sample average height:

```
170 cm
```

Using statistical calculations we obtain the range:

```
168 cm – 172 cm
```

Interpretation:

We say that the **true average height of all students is likely between 168 cm and 172 cm**.

This range is called a **confidence interval**.

---

# 4. Sample Variance Formula

Variance measures how spread out the data values are.

Formula for **sample variance**:

```
s^2 = Σ(xi − x̄)² / (n − 1)
```

Where:

| Symbol | Meaning               |
| ------ | --------------------- |
| s²     | Sample variance       |
| xi     | Individual data value |
| x̄     | Sample mean           |
| n      | Sample size           |

Variance helps estimate **population variability** in inferential statistics.

---

# 5. Confidence Interval (CI)

## Definition

A **confidence interval** is a **range of values that likely contains the true population parameter**.

Instead of reporting only a single number, statistics provides a **range with an associated confidence level**.

---

## Example

Suppose we estimate the **average salary of employees**.

Sample mean salary:

```
₹50,000
```

95% confidence interval:

```
₹47,000 – ₹53,000
```

Interpretation:

We are **95% confident that the true population average salary lies between ₹47,000 and ₹53,000**.

---

## Key Idea

| Type                | Description      |
| ------------------- | ---------------- |
| Point Estimate      | Single number    |
| Confidence Interval | Range of numbers |

---

# 6. Margin of Error (MOE)

## Definition

The **margin of error** represents the **maximum expected difference between the sample estimate and the true population value**.

It indicates **how precise an estimate is**.

---

## Example

Sample mean:

```
50
```

Confidence interval:

```
47 – 53
```

Margin of error:

```
3
```

Because:

```
50 − 3 = 47
50 + 3 = 53
```

The estimate is written as:

```
50 ± 3
```

---

## Formula Structure

```
Confidence Interval = Sample Estimate ± Margin of Error
```

Example:

```
50 ± 3
```

---

# 7. Significance Value (Significance Level, α)

## Definition

The **significance level (α)** represents the **probability of making an incorrect decision in statistical hypothesis testing**.

Specifically, it is the probability of **rejecting a true null hypothesis**.

---

## Common Significance Levels

| Significance Level | Interpretation    |
| ------------------ | ----------------- |
| 0.10               | 10% risk of error |
| 0.05               | 5% risk of error  |
| 0.01               | 1% risk of error  |

The most commonly used level is:

```
α = 0.05
```

Meaning:

There is a **5% chance of making a wrong statistical decision**.

---

## Example

Suppose we test whether a **new medicine works**.

If:

```
α = 0.05
```

It means there is a **5% chance that we conclude the medicine works when it actually does not**.

---

# 8. Relationship Between Key Concepts

| Concept                | Meaning                                     |
| ---------------------- | ------------------------------------------- |
| Confidence Interval    | Range where the true value likely lies      |
| Margin of Error        | Possible deviation from the true value      |
| Significance Level (α) | Probability of making an incorrect decision |

---

## Mathematical Relationship

```
Confidence Level = 1 − α
```

Example:

If:

```
α = 0.05
```

Then:

```
Confidence Level = 95%
```

---

# 9. Simple Real‑Life Example

Suppose a survey estimates that:

```
60% of people like a product
```

Margin of error:

```
±4%
```

Confidence interval:

```
56% – 64%
```

Confidence level:

```
95%
```

---

## Interpretation

We are **95% confident that the real percentage of people who like the product lies between 56% and 64%**.
