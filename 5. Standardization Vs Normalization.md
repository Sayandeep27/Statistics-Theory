# Standardization vs Normalization (Feature Scaling)

In **Machine Learning and Statistics**, **Standardization** and **Normalization** are techniques used to **scale numerical data** so that different features can be compared fairly.

Many algorithms (like **KNN, SVM, Linear Regression, Neural Networks**) work better when the data is on a **similar scale**.

---

# 1. Why Do We Need Scaling?

Imagine a dataset with two features:

| Feature | Example Values          |
| ------- | ----------------------- |
| Age     | 20, 30, 40              |
| Salary  | 20,000, 80,000, 120,000 |

Here:

* **Age range** → 20–40
* **Salary range** → 20,000–120,000

If we train a model:

* Salary will dominate the model because its numbers are **much larger**.

To avoid this problem we **scale the data**.

Two main scaling methods:

1. **Normalization**
2. **Standardization**

---

# 2. Normalization (Min–Max Scaling)

## Idea

**Normalization rescales values between 0 and 1.**

Formula:

```
X_norm = (X - X_min) / (X_max - X_min)
```

Where:

* **X** = original value
* **X_min** = minimum value in dataset
* **X_max** = maximum value in dataset

---

## Example

Dataset:

| Value |
| ----- |
| 10    |
| 20    |
| 30    |
| 40    |
| 50    |

Here:

* **Min = 10**
* **Max = 50**

Now normalize **30**.

```
X_norm = (30 - 10) / (50 - 10)
```

```
= 20 / 40
```

```
= 0.5
```

So:

| Original | Normalized |
| -------- | ---------- |
| 10       | 0          |
| 20       | 0.25       |
| 30       | 0.5        |
| 40       | 0.75       |
| 50       | 1          |

Now the values are **between 0 and 1**.

---

## Visualization Idea

Before normalization:

```
10 ----- 20 ----- 30 ----- 40 ----- 50
```

After normalization:

```
0 ----- 0.25 ----- 0.5 ----- 0.75 ----- 1
```

---

## Advantages

* Keeps data in a **fixed range (0–1)**.
* Works well for **distance-based algorithms**.

Example algorithms:

* **KNN**
* **K-Means**
* **Neural Networks**

---

# 3. Standardization (Z-score Scaling)

## Idea

Standardization transforms data so that:

* **Mean = 0**
* **Standard Deviation = 1**

Formula:

```
Z = (X - μ) / σ
```

Where:

* **X** = original value
* **μ (mu)** = mean of dataset
* **σ (sigma)** = standard deviation

This is called the **Z-score**.

---

## Example

Dataset:

| Value |
| ----- |
| 10    |
| 20    |
| 30    |
| 40    |
| 50    |

### Step 1 — Mean

```
Mean = (10 + 20 + 30 + 40 + 50) / 5
```

```
Mean = 30
```

---

### Step 2 — Standard Deviation

Standard deviation ≈ **14.14**

---

### Step 3 — Standardize value 40

```
Z = (40 - 30) / 14.14
```

```
Z ≈ 0.71
```

So the **standardized value of 40 ≈ 0.71**

---

### Approx Standardized Dataset

| Original | Standardized |
| -------- | ------------ |
| 10       | -1.41        |
| 20       | -0.71        |
| 30       | 0            |
| 40       | 0.71         |
| 50       | 1.41         |

Notice:

* Mean = **0**
* Values can be **negative or positive**

---

# 4. Key Difference

| Feature      | Normalization                       | Standardization                     |
| ------------ | ----------------------------------- | ----------------------------------- |
| Range        | 0 to 1                              | No fixed range                      |
| Based on     | Min and Max                         | Mean and Std Dev                    |
| Distribution | Does not assume normal distribution | Works best with normal distribution |
| Outliers     | Sensitive                           | Less sensitive                      |
| Result       | Scales to fixed interval            | Centers data around mean            |

---

# 5. Intuition (Very Important)

Think of **exam marks**.

### Normalization

Converts marks to a **percentage scale**.

Example:

| Marks  | Normalized |
| ------ | ---------- |
| 30/100 | 0.3        |
| 70/100 | 0.7        |

Everything becomes **0–1 scale**.

---

### Standardization

Tells **how far a student is from the average**.

Example:

| Marks | Z-score Meaning |
| ----- | --------------- |
| 0     | Exactly average |
| +1    | Above average   |
| -1    | Below average   |

So it measures **distance from mean**.

---

# 6. Python Implementation

## Normalization

```python
from sklearn.preprocessing import MinMaxScaler
import numpy as np

data = np.array([[10], [20], [30], [40], [50]])

scaler = MinMaxScaler()

normalized = scaler.fit_transform(data)

print(normalized)
```

Output:

```
[[0.  ]
 [0.25]
 [0.5 ]
 [0.75]
 [1.  ]]
```

---

## Standardization

```python
from sklearn.preprocessing import StandardScaler
import numpy as np

data = np.array([[10], [20], [30], [40], [50]])

scaler = StandardScaler()

standardized = scaler.fit_transform(data)

print(standardized)
```

Output:

```
[[-1.41]
 [-0.71]
 [ 0.  ]
 [ 0.71]
 [ 1.41]]
```

---

# 7. When to Use What (Important for ML)

Use **Normalization** when:

* Data has **no Gaussian distribution**
* Using **distance-based algorithms**

Examples:

* **KNN**
* **K-Means**
* **Neural Networks**

---

Use **Standardization** when:

* Data is approximately **normally distributed**
* Using algorithms that assume normality.

Examples:

* **Linear Regression**
* **Logistic Regression**
* **SVM**
* **PCA**

---

# 8. Simple Summary

**Normalization**

* Scales data between **0 and 1**
* Uses **min and max**

**Standardization**

* Converts data to **mean = 0**
* Uses **mean and standard deviation**
